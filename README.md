# Hubitat Project Template

This repository is a template for Hubitat apps and drivers that are shared publicly **as-is**, without support or ongoing maintenance.

It is intended for small, self-contained projects that were developed for personal use and later published for the benefit of others.

---

## Scope

Projects created from this template typically:

- Are designed for the Hubitat Elevation platform
- Consist of a single app, driver, or small app/driver pair
- Are feature-complete at the time of release
- Prioritise clarity and reliability over extensibility

---

## Non-goals

Projects created from this template are **not** intended to:

- Provide user support or troubleshooting
- Accept feature requests
- Guarantee compatibility with future Hubitat firmware versions
- Maintain backward compatibility indefinitely

---

## Support & Maintenance

No support is provided.

These projects are shared for reference and use at your own risk.  
There is no commitment to fix bugs, answer questions, or provide updates.

If you wish to modify or extend the code, please fork the repository and maintain your own copy.
