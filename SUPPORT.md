## Support Policy

No support is provided for projects created from this template.

These repositories are shared publicly for reference and personal use, exactly as published. There is no commitment to:

- Answer questions
- Provide troubleshooting assistance
- Accept feature requests
- Fix bugs
- Review pull requests
- Maintain compatibility with future Hubitat firmware releases

If the code does not meet your needs, you are encouraged to fork the repository and maintain your own copy.

Use of the code is entirely at your own risk.
